### 3. 간단한 문법 설명

*   `<!DOCTYPE html>` : HTML5 버전으로 작성되었음을 알리는 선언문
*   `<html>…</html>` : HTML 문서의 시작과 끝을 알리는 요소. 모든 태그(head, body)는 이 태그 안에 포함되어야 함
*   `<head>…</head>` : 문서의 메타데이터(인코딩, 제목 등)를 포함하는 부분
*   `<meta charset=”UTF-8”>` : 문자 인코딩 방식 설정
*   `<title>…</title>` : 브라우저 탭 창에 표시되는 제목
*   `<h2>…</h2>` : 제목을 나타내는 태그 (`<h1>`~`<h6>` 중 두 번째 크기)
*   `<a href=”…”>…</a>` : 하이퍼링크를 만드는 태그
*   `<br>` : 줄바꿈 태그
